# gitmeup

cli tool to automate git commands (add ., commit, pull push). 

### Installation :
Using npm 
```sh
$ npm install gitmeup -g
```
To update the package 
```sh
$ npm update gitmeup -g
```
### Usage guide : 
Run it inside any git repository:
```sh
$ gitmeup run
```
Change the default settings:
```sh
$ gitmeup defaults
```
