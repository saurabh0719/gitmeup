# gitmeup

cli tool to automate git commands.

<img src="demo.PNG"/>
  </p>

### Installation :
Using npm 
```sh
$ npm install gitmeup -g
```
To update the package 
```sh
$ npm update gitmeup -g
```
### Usage guide : 
Run it inside any git repository:
```sh
$ gitmeup run
```
Change the default settings:
```sh
$ gitmeup defaults
```
